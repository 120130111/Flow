PX4FLOW Board Firmware

Project:
http://px4.io/modules/px4flow

Dev guide / toolchain installation:
http://px4.io/dev/px4flow

To build, run:
```
  make
```

To flash via the PX4 bootloader (first run this command, then connect the board):
```
  make upload-usb
```
